# 文明六城市舒适度信息检索与开城决策支持系统

一个基于 React、Vite、TypeScript 和 Tailwind CSS 的课程展示原型项目。

系统围绕《文明 VI》开城决策场景，提供候选地块检索、条件筛选、排序、评分分析、地块详情和位置对比功能，体现“查询 → 筛选 → 排序 → 结果 → 分析 → 决策”的信息检索流程。

## 功能

- 信息检索工作台：关键词检索、多条件筛选、多维排序
- 检索结果解释：命中关键词高亮、匹配原因、匹配标签
- 城市评分仪表盘：综合评分、分项指标评分条、雷达图占位
- 城市详情分析：地图插图、基础信息、优势分析、风险分析、开城建议
- 建城位置对比：双地块选择、评分差异、产出差异
- 本地 mock 数据：26 个候选建城点

## 运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
